
This diretory contains scripts that have been used to produce the results presented in :
Druot et al., "A Multidisciplinary Airplane Research Integrated Library
With Applications To Partial Turboelectric Propulsion", AIAA Aviation conference, Dallas 2019

Only version 1.0.0 allows to reproduce exactly results of the above paper, next versions will introduce
some modifications that will shift a bit the results but the general outcomes are unchanged
